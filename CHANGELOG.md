# Changelog

## [0.1.1]
- Добавлена функциональность создания FB типов: `Basic FB` и `Simple FB`.
- Улучшено начальное отображение диаграмм: адаптивное масштабирование и авто-fit для схем разного размера.
- Добавлено отображение значений параметров у портов на canvas.
- Для входных data-портов добавлены дефолтные значения по типам (включая BOOL и TIME/DATE типы), если параметр не задан.

## [0.1.0]
- Диалог настроек плагина с сохранением параметров.
- Редактирование параметров функциональных блоков и OPC mapping.
- Удаление блоков, удаление и выделение соединений.
- Сохранение изменений диаграммы обратно в `.sys`.
- Генерация `.fboot` и deploy в OpenFB runtime.


## [0.0.1]
- Первоначальный релиз.
- Визуальный рендеринг диаграмм.
- Интерактивный canvas.
- Отладочное логирование.
