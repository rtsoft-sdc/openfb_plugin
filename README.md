# OpenFB Plugin для VS Code

Визуальный редактор диаграмм для IEC 61499 Function Block приложений.

## Обзор

OpenFB Plugin предоставляет интерактивный визуальный редактор для диаграмм IEC 61499 Function Block (FBD). Он интегрируется в VS Code для отображения файлов `.sys` (системных) как интерактивных диаграмм


## Возможности

### Текущие (версия 0.0.1)
- Открытие и визуализация файлов `.sys` диаграмм
- Интерактивный canvas с управлением масштабированием
- Отображение типа каждого блока над его визуализацией
- Визуализация соединений между блоками
- Логирование
- Русская локализация интерфейса

## Требования к системе

- **VS Code:** 1.85.0 или выше
- **Node.js:** 16.0.0 или выше (для сборки)
- **TypeScript:** 5.3.0+ (включена в зависимости)
- **ОС:** Windows или Linux (протестировано на Windows 10+)

## Установка и настройка

### 1. Клонирование и установка зависимостей

```bash
# Клонируем репозиторий
git clone <repository-url>
cd openFbPlugin

# Устанавливаем зависимости
npm install
```

## Сборка

### Сборка для разработки

```bash
# Компилируем TypeScript в JavaScript (backend + webview)
npm run compile
```
### Запуск в режиме отладки
```bash
# Press F5 in VS Code (or Run → Start Debugging)
```
